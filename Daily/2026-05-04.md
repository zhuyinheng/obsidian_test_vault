# MyStickies demo

This note is open in a **sticky window** — always on top, even over other apps.

## 1. Today's note, instantly

- Ribbon icon → opens today's sticky
- `Cmd+P` → **MyStickies: Open today's sticky window**
- Auto-opens whenever Obsidian launches

## 2. Any note → sticky

- Right-click any file → **Open as sticky**
- `Cmd+P` → **MyStickies: Open current note as sticky**
- Try it on [[Dashboard]] or [[Projects/Launch/Home|Launch home]]

## 5. Window controls

- Drag a corner to resize → MyStickies remembers the size for next time
- `Cmd+P` → **MyStickies: Close all sticky windows**
- "Go to main" button in the title bar focuses the main window

## 6. Multiple stickies at once

Right-click a few notes → **Open as sticky** to stack several side by side.
